# file pobo structure
## root key rule
key=value \
key name= any value \
Ex: \
hello= love mom \
world= is our place for living
## sub key rule
key= value > subkey1= sub value \
key= value >> subkey1= sub value \
key= value >>> subkey1= sub value \
Ex: \
dog = is animal who stay close with us \
container > \
panel1= a small container \
panel2= is bigger than spot \
